Text-Tags

In HTML kann man so gut wie alles darstellen, was man möchte. Einige Sachen sind einfach, andere etwas komplizierter. Oft verwendet man dafür z.B. CSS oder JavaScript. Mit dem Tag <p> kann man Texte schreiben.
Hier zeigen wir dir, wie du Texte in HTML formatieren kannst. Hier sind ein paar Beispiele:

TagFunktionGeschlossener Tag?
<b>Texte fett machenJa
<i>Texte kursiv machenJa
<u>Texte unterstreichenJa
<s>Texte durchstreichenJa

So kann man Überschriften schreiben:

TagFunktionGeschlossener Tag?
<h1>Größte ÜberschriftJa
<h2>Große ÜberschriftJa
<h3>Mittelgroße ÜberschriftJa
<h4>Kleine ÜberschriftJa
<h5>Sehr Kleine ÜberschriftJa
<h6>Kleinste ÜberschriftJa

Du kannst auch Kommentare in HTML mit dem Kommentar-Tag schreiben.
Zum Beispiel:

<!--Ich bin Ein Kommentar-->
