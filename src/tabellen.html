Tabellen

Tabellen sind eine andere Methode, Sachen in HTML zu strukturieren. Man kann sie so gestalten, wie man es haben möchte.
Eine Tabelle eröffnet man mit dem <table>-Tag und schließt sie wieder mit </table>.
Damit die Tabelle einen Rand hat, muss man das Attribut border="..." in das <table>-Tag einsetzen.
Das würde dann wie folgt aussehen: <table border="1">
Für die ... kann man jede Zahl einsetzen. Je höher die Zahl, desto dicker der Rand.
In einer Tabelle kann man die Anzahl der Reihen mit dem <tr>-Tag bestimmen. Nach dem <tr>-Tag
kann man die Anzahl der Spalten und was drin stehen soll, mit dem <td> oder <th> bestimmen
Danach schließt man wieder <td>/<th> und am Ende der Reihe schließt man <tr>
So würde das ganze dann im Quelltext aussehen:

<table border="1">
<tr><td>Blabla</td><td>Blabla</td></tr>
<tr><td>Blabla</td><td>Blabla</td></tr>
<tr><td>Blabla</td><td>Blabla</td></tr>
<tr><td>Blabla</td><td>Blabla</td></tr>
</table>

So würde das dann vom Browser angezeigt werden:

BlablaBlabla
BlablaBlabla
BlablaBlabla
BlablaBlabla


Mit dem colspan-Attribut für <td>/<th> kann man Spalten doppelt, dreifach, usw. so lang machen.

Mit dem rowspan-Attribut für <td>/<th> kann man Spalten doppelt, dreifach, usw. so tief machen.

So würde das ganze dann aussehen:


BlablaBlabla
BlablaBlabla
Blabla
BlablaBlablaBlabla

Und so würde es dann im Quelltext ausshen:


<table border="1">
<tr><td colspan="2">Blabla</td><td>Blabla</td></tr>
<tr><td rowspan="2" colspan="2">Blabla</td><td>Blabla</td></tr>
<tr><td>Blabla</td></tr>
<tr><td>Blabla</td><td>Blabla</td><td>Blabla </td><td>Blabla</td></tr>
</table>

Es kann auch eine Kopf-, Körper- und Fußzeile in eine Tabelle eigebaut werden.

Kopfzeile<thead>
Körperzeile<tbody>
Fußzeile<tfoot>

So würde das dann beispielsweise im Quelltext ausehen:


<table border="1">
<thead>
<tr>
<td>Blabla</td><td>Blabla</td>
<td>Blabla</td><td>Blabla</td>
</tr>
</thead>
<tbody>
<tr>
<td>Blabla</td><td>Blabla</td>
<td>Blabla</td><td>Blabla</td>
</tr>
<tr>
<td>Blabla</td><td>Blabla</td>
<td>Blabla</td><td>Blabla</td>
</tr>
</tbody>
<tfoot>
<tr>
<td>Blabla</td><td>Blabla</td>
<td>Blabla</td><td>Blabla</td>
</tr>
</tfoot>
</table>

Und so würde das ganze im Browser ausehen:


BlablaBlabla BlablaBlabla
BlablaBlabla BlablaBlabla
BlablaBlabla BlablaBlabla
BlablaBlabla BlablaBlabla
