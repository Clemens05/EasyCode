Einführung

HTML heißt ausgeschrieben "Hypertext Markup Language". Übersetzt heißt das "Hypertext Auszeichnungssprache".

Mit HTML kannst du deine eigenen Webseiten erstellen. Im Profi-Bereich wird zwar auch noch mit CSS (Cascading Style Sheets), JS (JavaScript) und mit anderen Werkzeugen gearbeitet, jedoch benötigt so gut wie jede Webseite HTML. HTML benutzt man oft nur zur Struktur der Webseite. Jedoch kannst auch du allein mit HTML deine eigene kleine Webseite machen.

HTML schreibt man mit sogenannten Tags. Sie werden geöffnet und oft geschlossen. Hier ein Beispiel:

<!DOCTYPE html>
<html>
<head>
<title>Webseite</title>
<head>
<body>
<p>Hello World!</p>
</body>
</html>

Eine HTML-Datei speichert man immer mit .html oder .htm ab.
